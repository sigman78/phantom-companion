This is index for source game data
==================================

Game reference
----------
 - "Phantom Epoch_Rulebook.pdf" - Main rulebook
 - manual/ - text extracted from rulebook pdf, somewhat garbled
 - TURN.md - Game turn explanation


Game data
---------

 - art/adv-class-actions - Adversaries Class Action cards
 - art/adv-species-actions - Adversaries Species Action cards
 - art/adv-icons - Adversaries icons
 - data - Game extracted data
 - data/adversaries.txt - list of game adversaries cards
 - data/class/ - adversaries class action cards in json
 - data/species/ - adversaries species action cards in json
 - data/adversary-stats.json - adversaries stats and crit specials (per diff level)
